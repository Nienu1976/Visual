# Changelog

## [Unreleased]

## [0.1.0] - 2026-03-30

### Added
- 機能①：漢字フラッシュクイズ生成（`visualquiz flash`）
  - ランダムブロック・横スキャン・縦スキャン・放射状の4モード
  - バッチ生成対応
- 機能②：歌詞漢字クイズ生成（`visualquiz lyrics`）
  - 漢字のみ表示 / 漢字以外のみ表示の2モード
  - ページ切り替えクロスフェードアニメーション
  - 正解フェードイン表示
- 機能③：漢字スライドパズル生成（`visualquiz puzzle`）
  - A*アルゴリズムによる最短手数解法
  - タイル移動イーズアウトアニメーション
  - 文字数からグリッドサイズを自動計算
- 共通機能
  - MP4エクスポート（H.264, 30fps, 1280×720 or 1920×1080）
  - PowerPoint (.pptx) への動画埋め込み
  - システムフォント自動検出・フォールバック
  - CLIインターフェース（`visualquiz` コマンド）
  - デモ生成コマンド（`visualquiz demo`）
  - 一括生成コマンド（`visualquiz all`）
